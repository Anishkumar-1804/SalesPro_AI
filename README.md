# 📈 SalesPro AI

SalesPro AI is a machine learning-based sales forecasting application that predicts future sales using historical sales data. Built with **Streamlit**, it provides an interactive dashboard for visualizing trends and generating forecasts.

## 🚀 Features

* 📊 Interactive Streamlit dashboard
* 📈 Sales forecasting with Prophet
* ⚡ Machine learning models using XGBoost and Ridge Regression
* 📅 Historical sales trend visualization
* 🔮 Future sales prediction
* 📁 CSV dataset upload support
* 📉 Model performance comparison

## 🛠️ Tech Stack

* Python
* Streamlit
* Prophet
* XGBoost
* Ridge Regression (Scikit-learn)
* Pandas
* NumPy
* Matplotlib

```bash
git clone https://github.com/your-username/SalesPro-AI.git
cd SalesPro-AI
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Workflow

1. Upload or load the sales dataset.
2. Preprocess the data.
3. Train forecasting models.
4. Generate future sales predictions.
5. Compare model performance.
6. Visualize forecasts through the dashboard.
